
# Short URL Generator 

Give your website your or any web url it will make short url for you.



## Installation

Install my-project with npm

```bash
  npm install short_url
  cd short_url
```
    
## API Reference

#### Get all items

```http
  POST /url
```

| Type     | Description                |
| :------- | :------------------------- |
| `json` | "{
"url": "www.arcadexit.com"
}"
